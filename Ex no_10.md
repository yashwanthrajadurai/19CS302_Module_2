EX NO 10 : C PROGRAM TO PRINT THE STRING "KEYBOARD" N NUMBER OF TIMES

AIM:

To write a program to print the string "KEYBOARD" n number of times.

ALGORITHM:

1. Start.
2. Declare the variables i,n.
3. Prompt the user to enter a value.
4. Read the value using scanf.
5. Run the for loop for “i” variable.
6. Print “KEYBOARD\n”.
7. End.

PROGRAM:

#include<stdio.h>

int main()

{

int i,n;

scanf("%d",&n);

for(i=1;i<=n;i++)

{
printf("KEYBOARD\n");

}
return 0;

}
